# 🎉 MATLAB Functions & Recursion Examples

## 📚 What’s inside?

### 1️⃣ Basic Addition Function
- `addNumbers(x,y)`  
  Simple function returning the sum of two numbers.

### 2️⃣ Factorial Calculations
- Iterative factorial using a `for` loop  
- Factorial with a custom function and loop  
- Recursive factorial function `factorial_n(n)`

### 3️⃣ Constant & Greeting Functions
- `getPi()` returns a fixed value of π  
- `greet(name)` displays a friendly hello message  
- `myFunction()` a no-input, no-output function that prints a message

### 4️⃣ Multiple Outputs Function
- `opeartions(x,y)` returns sum, subtraction, and multiplication of two numbers simultaneously
